# BecoSoft.FrontendTest

ASP.NET Core MVC project running on .NET 10.

## Prerequisites

- [.NET 10 SDK](https://dotnet.microsoft.com/download/dotnet/10.0)

## Build

```bash
dotnet build
```

## Run

```bash
dotnet run --project src/BecoSoft.FrontendTest.Web
```

The application will start at `http://localhost:5000` by default.

## Frontend Dependencies

This project is prepared to use:

- Bootstrap 4
- jQuery
- Font Awesome

Add them via LibMan, npm, or CDN in the layout file.
